# The Math behind Linear Regression 

Understanding linear regression is the gateway to grasping the fundamentals of machine learning. It’s not just about fitting a line to data; it’s about understanding the underlying mathematics that make this possible. In this post, we’ll break down the math behind linear regression, step by step.

## Goal

We are given a dataset of \( n \) points, each with \( m \) features. Our goal is to find a linear function that best predicts the output variable \( y \) based on the input features \( x_1, x_2, ..., x_m \).
