About This Demo

This is a 3D snake game built using modern web technologies:

  • Three.js - WebGL renderer for 3D graphics
  • Cannon.js - Physics engine for realistic movement
  • CSS2DRenderer - For HUD overlays

How to Play

  • Desktop: Use arrow keys (←, ↑, →) to control the snake
  • Mobile: Tap and hold to move, swipe left/right to turn

The game features a camera that follows the snake, realistic physics simulation, and responsive controls that adapt to your device type.
